# AI 日报：2026年04月15日 | 巨头竞速升级，AI从工具走向自主代理

今日AI领域呈现出一个清晰的主线：**AI正在从被动工具向自主代理演进**。Anthropic发布Claude Code Routines让AI代理实现自动化工作流，Google将Gemini提示词变成可复用的"技能"，Meta甚至开始打造CEO的AI数字替身。与此同时，AI引发的社会争议也在升级——针对Sam Altman住所的袭击事件敲响了警钟。让我们逐一解读今天的5个重要AI新闻。

---

## 🚀 Anthropic发布Claude Code Routines：AI编程代理进入自动化时代

**事件概要**：Anthropic正式推出Claude Code Routines功能，允许开发者将Claude Code配置为自动化代理，通过定时触发、API调用或GitHub事件来自动执行编程任务。该功能在Hacker News上获得414点热度，引发257条讨论。

**关键细节**：

- **三种触发模式**：Routines支持定时触发（每小时/每夜/每周）、API触发（通过HTTP POST端点调用）和GitHub事件触发（PR、push、issue等），单一Routine可组合多种触发器
- **云端执行**：Routines在Anthropic托管的云基础设施上运行，即使笔记本电脑关机也能持续工作
- **MCP连接器支持**：Routines可搭配Claude Code的连接器（MCP）使用，扩展与外部工具的集成能力
- **多计划覆盖**：功能面向Pro、Max、Team和Enterprise用户开放，目前处于研究预览阶段
- **桌面端同步改版**：The Verge报道Claude Code桌面应用也进行了重新设计，新增侧边栏管理多会话、拖拽布局自定义工作区、内置终端和文件编辑器

**深度分析**：

Claude Code Routines的发布标志着AI编程工具从"交互式助手"向"自主代理"的关键跃迁。过去，开发者需要手动启动Claude Code、输入提示词、等待结果。现在，AI可以在特定事件发生时自动介入——比如每次有新PR时自动进行代码审查，或每天定时运行测试套件。

这一趋势与Google在Chrome中推出的"Skills"功能遥相呼应（见下文），反映出整个行业正在将AI从"一次性对话"升级为"可编排的工作流"。对于软件开发团队而言，这意味着AI不再是锦上添花的工具，而可能成为CI/CD流水线中的标准组件。

值得注意的是，Anthropic同时发布了桌面端的全面改版，将多会话管理、内置终端等开发者高频需求整合进来。这表明Anthropic正在将Claude Code打造为一个完整的开发环境，而不仅仅是一个聊天界面。

## ⚠️ 针对Sam Altman住所的袭击事件：AI产业安全警钟敲响

**事件概要**：据The Verge报道，一名20岁男子涉嫌向OpenAI CEO Sam Altman住所投掷燃烧瓶，两天后其住所疑似再次遭到针对。同期，印第安纳波利斯一名市议员因支持数据中心开发项目，家门被射击13发子弹，现场留下"No Data Centers"字条。

**关键细节**：

- **嫌疑人动机**：据《旧金山纪事报》调查，嫌疑人此前曾撰文表达对AI竞赛可能导致人类灭绝的恐惧
- **行业反应**：多个反AI加速发展组织（包括StopAI和PauseAI）明确谴责暴力行为，强调绝大多数AI抗议是非暴力的
- **背景升级**：过去几年，围绕数据中心的本地抗议持续升级，密歇根州曾有蒙面抗议者到公用事业官员家中捣毁打印机
- **Altman的回应**：袭击事件发生后，Altman似乎部分归咎于此前《纽约客》发表的长篇调查报道对他的负面刻画
- **更广泛的张力**：AI行业面临的阻力包括就业替代担忧、气候影响、缺乏安全护栏的快速发展等多个层面

**深度分析**：

这一事件揭示了AI产业高速扩张背后的社会张力正在达到临界点。从经济层面看，AI的快速渗透正在引发真实的社会焦虑——无论是对失业的恐惧，还是对数据中心能源消耗的不满。当这种焦虑无法通过正常渠道有效释放时，极端行为的风险就会增加。

从产业治理的角度看，这些事件可能会推动AI公司在公众沟通和社会责任方面投入更多资源。同时，政府层面的AI监管讨论也可能因此加速。对于整个行业而言，如何在追求技术突破的同时赢得公众信任，已经成为一个不可回避的命题。

## 🔧 Google Chrome推出"Skills"功能：将Gemini提示词变成一键复用技能

**事件概要**：Google在Chrome浏览器中推出"Skills"功能，用户可以将常用的Gemini AI提示词保存为可复用的"技能"，通过斜杠命令（/）或点击按钮一键触发。该功能今日开始推送，免费向所有Chrome用户开放。

**关键细节**：

- **保存与复用**：用户可以将任何有效的Gemini提示词保存为Skill，在任何网页中通过输入"/"快速调用
- **跨设备同步**：只要登录Google账号，保存的Skills可以在不同设备间同步
- **预置技能库**：Google同时推出Skills Library，提供预制的Gemini提示词供用户选择和编辑
- **安全机制不变**：Skills遵循与手动输入相同的安全规则，涉及日历或消息发送等敏感操作时仍需确认
- **模型选择自由**：用户可选择Pro模型（更准确但更慢）或Fast模型（更快但可能出错）

**深度分析**：

Chrome Skills看似是一个小功能更新，但其背后的战略意图值得关注。Google正在通过将AI能力"碎片化"并嵌入浏览器的日常使用场景中，试图培养用户的AI使用习惯。当用户发现"保存提示词→一键触发"比"每次重新输入"方便得多时，Gemini在Chrome中的使用频率将显著提升。

这与Anthropic的Claude Code Routines形成了有趣的对照：两者都在将AI从"单次对话"模式升级为"可复用的工作单元"，只是Anthropic瞄准的是开发者场景，而Google瞄准的是普通用户的浏览器使用场景。可以预见，"提示词工程"将逐步演变为"技能编排"，成为AI时代的新基本功。

## 🛡️ 英国政府测试Anthropic Mythos：首个完成多步渗透挑战的AI模型

**事件概要**：英国AI安全研究所（AISI）发布了对Anthropic Mythos Preview模型的网络安全能力评估。测试结果显示，Mythos成为首个能够从头到尾完成"最后的防线"（The Last Ones）32步数据提取攻击挑战的AI系统。

**关键细节**：

- **The Last Ones挑战**：AISI设计的32步渗透测试，模拟对企业网络的数据提取攻击，人类专家估计需要约20小时才能完成
- **Mythos表现**：在10次尝试中成功3次完成全程攻击，平均每次运行可完成32步中的22步，显著优于Claude 4.6的16步
- **横向对比**：在基础CTF（夺旗赛）任务中，Mythos与GPT-5.4、Claude Opus 4.6等模型差距不大（准确率相差5-10%），其独特优势在于多步骤任务链的能力
- **仍有局限**：Mythos在更难的"Cooling Tower"测试（模拟攻击发电厂控制系统）中仍表现不佳
- **AISI建议**：随着未来模型能力匹配或超越Mythos，系统防御者应同样利用AI模型来加固防御

**深度分析**：

Mythos测试结果的意义在于揭示了AI网络安全能力的一个关键拐点：**单步任务能力已接近饱和，真正的突破在于多步骤任务链的编排能力**。这意味着AI不再只是辅助人类黑客完成单个攻击步骤的工具，而是可能独立执行复杂渗透流程的自主代理。

从商业角度看，这将催生一个规模可观的"AI红队"市场——企业需要使用同等能力的AI来测试自身防御。同时，这也对全球网络安全基础设施提出了新的挑战：当攻击可以被AI自动化执行时，防御的响应速度和复杂度也必须同步提升。AISI的建议非常明确：用AI对抗AI。

## 🤖 Meta打造扎克伯格AI数字替身：CEO亲自训练自己的虚拟形象

**事件概要**：据英国《金融时报》报道，Meta正在开发一个AI驱动的、照片级真实的3D版扎克伯格，可用于与员工互动交流。扎克伯格本人每周投入5-10小时参与编码和AI项目技术评审。

**关键细节**：

- **AI替身细节**：该角色将基于扎克伯格的图像、声音、行为举止和语调进行训练，并融入其对公司战略的最新思考
- **早期阶段**：项目仍处于早期开发阶段，旨在让员工通过与AI扎克伯格互动而感受到与创始人的连接
- **技术挑战**：照片级真实感的3D AI角色需要大量计算资源来实现实时交互，避免交互延迟
- **Muse Spark发布**：Meta同期发布了Muse Spark模型，专注于健康推理和视觉理解等高级能力，推动股价上涨7%
- **内部AI化**：Meta正在鼓励员工使用AI工具，产品经理被要求进行AI技能基线测试，包括"vibe coding"练习

**深度分析**：

Meta打造CEO AI替身的举动，表面上是一个大胆的技术实验，但深层逻辑值得玩味。首先，这延续了Meta在AI角色领域的布局——从2023年的名人AI聊天机器人，到AI Studio平台，再到现在的CEO数字替身。Meta显然认为，**个性化的AI交互体验**将是社交平台的下一个竞争焦点。

其次，扎克伯格亲自参与训练的行为传递了一个信号：AI不仅是Meta的产品方向，更是其内部运营的核心。当一家公司的CEO每周花10小时写代码并训练自己的AI版本时，这已经超越了技术探索的范畴，而是一种组织文化的重塑。

从更宏观的视角看，"AI数字替身"概念可能面临严重的社会和伦理争议。当AI可以完美模拟一个人的外貌、声音和思维方式时，我们如何定义"真实的人际互动"？这将是Meta在推进这一项目时不得不面对的问题。

## 总结与展望

今天的AI新闻共同描绘了一幅**AI代理化浪潮**的图景：

1. **工具→代理**：Anthropic的Routines、Google的Skills，都在将AI从"用完即走"的工具升级为"持续运行"的代理
2. **能力边界扩展**：Mythos的多步渗透测试成功表明，AI自主执行复杂任务链的能力正在突破关键门槛
3. **人机关系重塑**：Meta的AI数字替身和扎克伯格的亲自参与，预示着企业内部人机协作模式的深层变革
4. **社会张力加剧**：针对AI行业领军人物的暴力事件提醒我们，技术创新必须与社会责任并行

展望未来，2026年很可能成为"AI代理元年"——从编程到安全，从办公到社交，AI将不再只是人类的工具，而逐渐成为可以独立行动的数字实体。如何在释放AI潜力的同时管理其风险，将是整个行业和社会需要共同面对的核心议题。
